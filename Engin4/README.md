# BartEngine

BartEngine est un engin de jeu 2D fait en C++ dans le cadre du cours de  programmation avancée II au collège Bart.

## Version 1.0.4

- Demo Tiled
- Fixed bugs

## Version 1.0.3

- Collision detection with maps
- Change value of tiles on a map

## Version 1.0.2

- Fixed memory leaks from version 1.0.1

## Version 1.0.1

- Support du 64/32 bit en Debug/Release
- La classe HashKey est remplacer par std::hash
- Rendu de texture supporte les flip horizontal et vertical.
- Tiled, multiple tileset: on peu maintenant utiliser plusieurs tileset dans une même carte.
- Tiled support les flip de tuiles.
- Tiled support pour rotation des tuiles.
- Ajouter une méthode pour rendre un rectangle plein
- Résoudre les fuites de mémoire avec Tiled.
- Ajouter un component pour faire des boutons
- Ajouter le culling avec le viewport de la camera et une map.

## Version 1.0.0

- Support pour le chargement de maps fait avec Tiled.
- Support de la manette de XBox.
- Support de la sourie.
- Système de caméra.
